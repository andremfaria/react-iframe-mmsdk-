# React + TypeScript + Vite

## How to run
Install the needed dependencies
```bash
pnpm install
```

Run the app
```bash
pnpm run dev
```

Open the app on on:
```bash
http://YOUR_IP_ADDRESS:5173/iframe/index.html
```
